# 日常运营

本文介绍门店日常运营流程，帮助经理高效管理门店日常工作。

<Video id="V09" title="日常运营视频教程" duration="4-5 分钟" />

## 适用角色

- 经理 (Manager)
- 管理员 (Admin)

## 开店流程

每天营业前需要完成的准备工作。

### 开店检查清单

<Steps>

### 1. 系统登录

使用经理账号登录系统，进入门店管理界面。

### 2. 查看今日概览

检查仪表板上的关键信息：
- 今日预约数量
- 待确认预约
- 员工出勤情况
- 系统通知

### 3. 确认员工出勤

查看已打卡员工，确认人手是否充足：
- 点击 **员工 > 考勤记录**
- 查看今日打卡状态
- 如有缺勤，安排调班

### 4. 检查预约情况

预览今日所有预约：
- 确认预约已分配技师
- 检查是否有时间冲突
- 联系未确认的客户

### 5. 准备工位

确认服务工位准备就绪：
- 清洁和整理
- 耗材检查
- 设备测试

</Steps>

## 营业中管理

### 实时监控

在营业时间内，经理可通过仪表板实时监控：

| 监控项 | 说明 |
|--------|------|
| 当前客流 | 店内客人数量 |
| 服务进行中 | 正在进行的服务 |
| 等待队列 | 等待服务的客人 |
| 今日营收 | 实时销售额 |

### 预约调度

**处理新预约**
1. 查看新预约请求
2. 确认时间可用性
3. 分配合适的技师
4. 发送确认通知

**处理 Walk-in 客人**
1. 检查当前可用技师
2. 评估等待时间
3. 如需等待，告知预计时间
4. 创建临时预约

### 技师调配

根据实际情况调整技师分配：

- **繁忙时段** - 优化技师利用率
- **空闲时段** - 安排休息或培训
- **突发状况** - 临时调配支援

### 客户服务

处理客户反馈和特殊情况：

- 客户投诉处理
- 服务补救措施
- VIP 客户接待
- 紧急情况处理

## 预约看板

### 看板视图

经理可使用预约看板全览当日情况：

- **日视图** - 按时间轴显示所有预约
- **周视图** - 查看一周预约分布
- **员工视图** - 按员工分列显示

### 看板操作

在看板上可直接进行：

- **拖拽调整** - 调整预约时间或技师
- **快速编辑** - 修改预约信息
- **状态更新** - 标记签到、完成等
- **新建预约** - 直接在空白时段创建

<Callout type="info">
看板支持实时同步，多人同时操作时会自动更新。
</Callout>

## 员工管理

### 考勤监控

- 查看员工打卡记录
- 处理迟到早退
- 审批请假申请
- 安排临时调班

### 业绩追踪

实时查看员工当日业绩：

- 服务数量
- 营业额
- 小费收入
- 客户评价

### 任务分配

为员工分配额外任务：

- 工位清洁
- 库存盘点
- 培训学习
- 其他指派

## 异常处理

### 预约变更

**客户取消**
1. 确认取消原因
2. 处理预付款（如有）
3. 释放时间段
4. 更新技师安排

**客户迟到**
1. 联系客户确认
2. 评估是否影响后续预约
3. 决定等待或重新安排

**客户爽约 (No-show)**
1. 尝试联系客户
2. 记录爽约
3. 释放时间段
4. 考虑收取违约费（如政策允许）

### 服务问题

**客户不满意**
1. 耐心倾听客户意见
2. 评估问题严重程度
3. 提供解决方案（重做/折扣/退款）
4. 记录问题避免再次发生

**服务超时**
1. 通知后续客户可能延迟
2. 调整排班或增派人手
3. 向受影响客户致歉

### 设备故障

1. 评估影响范围
2. 启用备用设备（如有）
3. 调整服务安排
4. 联系维修或支持

## 数据看板

### 今日数据

| 指标 | 说明 |
|------|------|
| 预约完成率 | 完成/总预约 |
| 人均产值 | 营业额/员工数 |
| 平均等待时间 | 客人平均等待时长 |
| 客户满意度 | 当日评价分数 |

### 对比分析

与历史数据对比：
- 同比昨日
- 同比上周同日
- 达成目标进度

## 闭店准备

营业结束前的收尾工作。

<Steps>

### 1. 确认最后服务

确保所有服务完成并结账。

### 2. 检查待处理事项

- 未完成的客户跟进
- 次日需要准备的事项
- 未处理的问题

### 3. 进行日结

完成每日结算流程（详见日结操作）。

### 4. 系统检查

- 确认所有数据已保存
- 检查系统通知
- 预览次日预约

### 5. 安全检查

- 设备关闭
- 门店安全
- 系统登出

</Steps>

## 常见问题

<FAQ>
  <FAQItem question="临时有员工请假怎么办？">
    首先检查今日预约负载，如果可以由其他员工分担则重新分配。如果人手不足，需要联系部分客户改期，优先保留老客户和高价值订单。
  </FAQItem>
  <FAQItem question="系统突然无法使用怎么办？">
    先检查网络连接，如果网络正常则联系技术支持。紧急情况下可以使用纸笔记录，待系统恢复后补录。
  </FAQItem>
  <FAQItem question="如何处理客户之间的纠纷？">
    保持冷静和专业，将冲突双方分开处理。倾听各方诉求，公平合理地解决问题。必要时可以提供小礼品安抚情绪。
  </FAQItem>
</FAQ>
