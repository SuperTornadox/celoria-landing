# 排班管理

本文介绍如何使用 Celoria 管理员工排班，包括创建班次、排班计划、调班和请假处理。

<Video id="V10" title="排班管理视频教程" duration="4-5 分钟" />

## 适用角色

- 超级管理员 (Super Admin)
- 管理员 (Admin)
- 经理 (Manager)

## 前置条件

- 已登录管理员或经理账号
- 拥有 `employees:schedule` 权限

## 排班概览

进入 **员工 > 排班管理** 查看排班情况。

### 视图模式

| 模式 | 说明 | 适用场景 |
|------|------|----------|
| 周视图 | 显示一周排班 | 日常管理 |
| 月视图 | 显示整月排班 | 整体规划 |
| 员工视图 | 按员工显示 | 查看个人排班 |
| 日视图 | 显示单日详情 | 当日调整 |

### 排班状态

| 颜色 | 状态 | 说明 |
|------|------|------|
| 绿色 | 正常班次 | 已确认的排班 |
| 黄色 | 待确认 | 等待员工确认 |
| 灰色 | 休息日 | 该日不排班 |
| 红色 | 请假 | 已批准的请假 |

## 班次模板

### 预设班次

系统提供常用班次模板：

| 班次名称 | 时间 | 说明 |
|----------|------|------|
| 早班 | 09:00-17:00 | 8小时工作制 |
| 晚班 | 13:00-21:00 | 8小时工作制 |
| 全天 | 09:00-21:00 | 12小时（含休息） |
| 半天上午 | 09:00-13:00 | 4小时 |
| 半天下午 | 13:00-17:00 | 4小时 |

### 自定义班次

<Steps>

### 1. 进入班次设置

点击 **排班管理 > 班次设置**。

### 2. 添加新班次

点击 **添加班次**。

### 3. 配置班次信息

| 字段 | 说明 |
|------|------|
| 班次名称 | 易于识别的名称 |
| 开始时间 | 上班时间 |
| 结束时间 | 下班时间 |
| 休息时段 | 午休或休息时间 |
| 班次颜色 | 在日历中的显示颜色 |

### 4. 保存班次

</Steps>

## 创建排班

### 单日排班

<Steps>

### 1. 选择日期

在排班日历中点击目标日期。

### 2. 选择员工

从员工列表中选择要排班的员工。

### 3. 选择班次

选择预设班次或自定义时间。

### 4. 保存排班

点击保存完成该员工当日排班。

</Steps>

### 批量排班

快速为多人多天创建排班。

<Steps>

### 1. 进入批量排班

点击 **批量排班** 按钮。

### 2. 选择员工

勾选需要排班的员工（可多选）。

### 3. 选择日期范围

设置排班的开始和结束日期。

### 4. 选择班次模式

- **固定班次** - 每天相同班次
- **轮班** - 按周期轮换
- **按周重复** - 每周相同安排

### 5. 预览并确认

预览排班结果，确认无误后保存。

</Steps>

### 周模板

使用周模板快速复制排班。

<Steps>

### 1. 创建周模板

将当前周的排班保存为模板。

### 2. 应用模板

选择目标周，应用已保存的模板。

### 3. 微调

根据需要调整个别日期。

</Steps>

<Callout type="info">
周模板会自动跳过节假日。节假日排班需要单独设置。
</Callout>

## 调班管理

### 员工申请调班

员工可以申请调班：

1. 员工在 App 中提交调班申请
2. 选择要调换的日期和班次
3. 提交申请说明原因

### 经理处理调班

<Steps>

### 1. 查看调班申请

进入 **排班管理 > 调班申请**。

### 2. 审核申请

查看申请详情：
- 申请人和原因
- 原班次和目标班次
- 对其他排班的影响

### 3. 批准或拒绝

- **批准** - 系统自动更新排班
- **拒绝** - 填写拒绝原因

</Steps>

### 经理直接调班

经理可以直接调整排班：

1. 在排班日历中找到目标班次
2. 拖拽到新的时间或员工
3. 或点击编辑修改班次详情
4. 保存更改

<Callout type="warning">
直接调班会通知受影响的员工。建议提前与员工沟通。
</Callout>

## 请假管理

### 请假类型

| 类型 | 扣薪 | 说明 |
|------|------|------|
| 年假 | 否 | 带薪年假 |
| 病假 | 视情况 | 需提供证明 |
| 事假 | 是 | 个人事务 |
| 调休 | 否 | 加班换休 |

### 员工请假流程

1. 员工在 App 中提交请假申请
2. 选择请假类型和日期
3. 填写请假原因
4. 等待经理审批

### 经理审批请假

<Steps>

### 1. 查看请假申请

进入 **排班管理 > 请假申请**。

### 2. 审核信息

检查：
- 请假时间和类型
- 该时段预约情况
- 其他员工排班情况

### 3. 处理决定

- **批准** - 自动更新排班，释放已分配的预约
- **拒绝** - 填写拒绝原因
- **需要调整** - 与员工沟通调整日期

</Steps>

### 请假影响

请假批准后：
- 该时段排班自动取消
- 已分配的预约需要重新安排
- 通知相关客户（如有预约）

## 考勤统计

### 月度考勤

查看员工月度考勤汇总：

| 统计项 | 说明 |
|--------|------|
| 应出勤天数 | 当月排班天数 |
| 实际出勤 | 正常打卡天数 |
| 迟到次数 | 迟到的天数 |
| 早退次数 | 早退的天数 |
| 请假天数 | 已批准请假 |
| 缺勤天数 | 未打卡且未请假 |

### 导出考勤

可导出考勤数据用于：
- 工资计算
- 绩效考核
- 劳动合规

## 排班规则

### 合规设置

配置排班合规规则：

- **最大连续工作天数** - 如不超过6天
- **两班之间最短间隔** - 如至少8小时
- **每周最大工时** - 如不超过40小时
- **加班提醒阈值** - 超时自动提醒

### 冲突检测

系统自动检测排班冲突：

- 同一员工时间重叠
- 超出合规限制
- 与请假时间冲突

<Callout type="error">
存在冲突的排班无法保存，需要先解决冲突。
</Callout>

## 常见问题

<FAQ>
  <FAQItem question="如何处理员工临时请假？">
    在排班页面直接将该员工当日班次改为请假状态。系统会自动释放该员工的预约，你需要手动重新分配给其他员工。
  </FAQItem>
  <FAQItem question="排班后员工能看到吗？">
    可以。员工在 App 中可以查看自己的排班计划。排班变更会通过推送通知员工。
  </FAQItem>
  <FAQItem question="可以设置跨店排班吗？">
    可以。如果员工分配了多个门店，可以在不同门店为其排班。系统会检测时间冲突。
  </FAQItem>
  <FAQItem question="如何设置节假日排班？">
    节假日排班需要单独设置。在系统设置中配置节假日，然后为需要上班的员工手动排班。
  </FAQItem>
</FAQ>
