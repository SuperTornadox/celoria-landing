# 日结准备

日结前需要完成的检查和准备工作。

## 适用角色

- 经理 (Manager)
- 管理员 (Admin)

## 前置条件

- 已登录经理或管理员账号
- 拥有 `closeout:manage` 权限
- 当日所有服务已完成结账

## 日结概述

日结是每天营业结束后的重要流程，用于：

- 核对当日营收
- 清点现金抽屉
- 确认账目无误
- 生成每日报表

<Callout type="warning">
建议每天固定时间进行日结，保持财务数据的准确性和一致性。
</Callout>

## 日结前检查

### 检查清单

开始日结前确认：

| 检查项 | 说明 |
|--------|------|
| 所有服务已完成 | 无进行中的服务 |
| 所有订单已结账 | 无未支付订单 |
| 预约状态更新 | 爽约/取消已标记 |
| 员工已下班 | 或知悉日结进行中 |

## 未完成事项处理

如有未完成事项：

**进行中的服务**
- 等待服务完成后再日结
- 或标记为次日继续

**未结账订单**
- 联系客户完成支付
- 或标记为欠款/挂账
