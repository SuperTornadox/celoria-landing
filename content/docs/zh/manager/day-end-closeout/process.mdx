# 日结流程

完成日结的详细操作步骤。

## 操作步骤

<Steps>

### 1. 进入日结页面

点击 **管理 > 日结** 或顶部快捷入口。

### 2. 系统自动汇总

系统自动计算当日数据：
- 总营收
- 各支付方式明细
- 服务数量统计
- 员工业绩汇总

### 3. 核对销售数据

检查系统显示的销售数据是否正确：

| 项目 | 说明 |
|------|------|
| 服务收入 | 今日服务总额 |
| 产品收入 | 产品销售总额 |
| 礼品卡销售 | 新售礼品卡金额 |
| 小费收入 | 今日小费总额 |
| 退款 | 今日退款总额 |

### 4. 清点现金

清点现金抽屉实际金额：

**现金明细输入**
- $100 × ___ 张 = $___
- $50 × ___ 张 = $___
- $20 × ___ 张 = $___
- $10 × ___ 张 = $___
- $5 × ___ 张 = $___
- $1 × ___ 张 = $___
- 硬币 = $___
- **合计 = $___**

### 5. 现金核对

系统自动比较：

| 项目 | 金额 |
|------|------|
| 期初现金 | 今日开始时的现金 |
| 今日现金收入 | 现金支付订单总额 |
| 今日现金支出 | 现金找零、退款等 |
| **应有现金** | 计算应有金额 |
| **实际现金** | 你清点的金额 |
| **差异** | 应有 - 实际 |

### 6. 处理差异

如果存在差异：

**金额小于阈值（如 $5）**
- 记录差异原因
- 可直接完成日结

**金额超过阈值**
- 必须填写差异说明
- 可能需要上级审批

### 7. 确认信用卡收入

核对信用卡/刷卡收入：
- 检查 POS 机结算金额
- 与系统显示金额比对
- 记录差异（如有）

### 8. 完成日结

确认所有数据无误后：
1. 点击 **完成日结**
2. 输入确认密码（如需要）
3. 日结完成，数据锁定

</Steps>

## 异常处理

### 账目不平

**找出差异原因**
1. 检查是否有遗漏的交易
2. 检查退款是否正确处理
3. 检查现金清点是否准确

**常见原因**
- 找零计算错误
- 忘记录入的交易
- 重复录入
- POS 机故障

### 强制日结

特殊情况下可以强制日结：
- 需要填写强制原因
- 需要更高权限确认
- 会记录在审计日志

<Callout type="error">
强制日结应该是最后手段。请尽量找出差异原因后再完成日结。
</Callout>

### 日结后修改

日结后数据通常锁定，如需修改：
1. 联系管理员
2. 说明修改原因
3. 管理员解锁后修改
4. 重新完成日结
