# 日结报表

日结完成后自动生成的报表。

## 报表内容

**销售汇总**
- 总营收
- 各类收入明细
- 折扣和退款

**支付方式明细**
- 现金
- 信用卡
- 礼品卡
- 储值余额
- 其他

**员工业绩**
- 每位员工的服务收入
- 小费分配
- 服务数量

## 报表操作

| 操作 | 说明 |
|------|------|
| 打印 | 打印纸质报表 |
| 导出 PDF | 保存电子档案 |
| 发送邮件 | 发送给管理层 |
