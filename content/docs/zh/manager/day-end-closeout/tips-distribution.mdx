# 小费分配

日结时确认小费的分配方式。

## 分配方式

根据门店政策，小费可以：

| 方式 | 说明 |
|------|------|
| 归属技师 | 小费全额归服务技师 |
| 按比例分配 | 技师和前台按比例分 |
| 池化分配 | 所有小费平均分配 |

## 确认分配

日结时确认小费分配：
1. 查看系统计算的分配方案
2. 如需调整，手动修改
3. 确认分配后锁定

<Callout type="info">
小费分配方式在系统设置中配置。日结时只能在规则范围内微调。
</Callout>
