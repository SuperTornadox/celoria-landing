# 团队管理

本文介绍门店经理如何使用 Celoria 进行团队管理，包括绩效追踪、任务分配和团队协作。

## 适用角色

- 经理 (Manager)
- 管理员 (Admin)

## 前置条件

- 已登录经理或管理员账号
- 拥有 `employees:view` 和 `employees:schedule` 权限

## 团队概览

### 团队仪表板

在团队仪表板查看整体情况：

| 指标 | 说明 |
|------|------|
| 在岗人数 | 当前已打卡员工 |
| 服务中 | 正在提供服务的员工 |
| 空闲 | 可接待新客的员工 |
| 今日业绩 | 团队整体营收 |

### 员工状态

实时查看每位员工状态：

| 状态 | 说明 |
|------|------|
| 在线 | 已打卡，空闲中 |
| 服务中 | 正在为客人服务 |
| 休息中 | 休息时间 |
| 离线 | 未打卡或已下班 |

## 绩效管理

### 业绩追踪

追踪员工的关键绩效指标：

**服务指标**
- 服务数量
- 服务营收
- 平均客单价
- 小费收入

**效率指标**
- 服务完成率
- 平均服务时长
- 客户等待时间

**质量指标**
- 客户评价分数
- 返客率
- 投诉次数

### 绩效排名

查看团队绩效排名：

| 维度 | 说明 |
|------|------|
| 营收排名 | 按服务收入排名 |
| 服务量排名 | 按服务数量排名 |
| 评价排名 | 按客户评分排名 |
| 小费排名 | 按小费收入排名 |

### 设置目标

为团队和个人设置目标：

<Steps>

### 1. 进入目标设置

点击 **团队 > 目标管理**。

### 2. 创建目标

设置目标类型：
- 门店整体目标
- 个人目标

### 3. 配置目标参数

| 字段 | 说明 |
|------|------|
| 目标名称 | 如「本月营收目标」 |
| 目标值 | 具体数值 |
| 时间周期 | 日/周/月 |
| 适用员工 | 全员或指定人员 |

### 4. 保存并启用

</Steps>

### 进度追踪

查看目标完成进度：
- 实时进度条
- 与历史同期对比
- 预测能否达成

## 任务分配

### 创建任务

为员工分配工作任务：

<Steps>

### 1. 创建新任务

点击 **团队 > 任务 > 新建任务**。

### 2. 填写任务信息

| 字段 | 说明 |
|------|------|
| 任务标题 | 简明的任务描述 |
| 任务详情 | 具体要求和说明 |
| 优先级 | 高/中/低 |
| 截止时间 | 完成期限 |
| 负责人 | 指定执行员工 |

### 3. 发布任务

任务会推送给负责人。

</Steps>

### 任务类型

| 类型 | 示例 |
|------|------|
| 日常任务 | 清洁工位、整理工具 |
| 培训任务 | 学习新技能、观看培训视频 |
| 特殊任务 | 接待VIP、筹备活动 |
| 盘点任务 | 库存盘点、设备检查 |

### 跟进任务

- 查看任务完成情况
- 提醒逾期任务
- 验收完成的任务
- 给予评价反馈

## 沟通协作

### 团队公告

发布团队公告通知所有成员：

1. 进入 **团队 > 公告**
2. 点击 **发布公告**
3. 填写公告内容
4. 选择发送范围（全员/指定人）
5. 发布后推送通知

### 公告类型

- **普通公告** - 一般通知
- **重要公告** - 需确认已读
- **紧急公告** - 立即推送

### 消息通知

与员工进行一对一沟通：
- 发送文字消息
- 查看已读状态
- 消息历史记录

## 培训管理

### 培训计划

为团队安排培训：

<Steps>

### 1. 创建培训

点击 **团队 > 培训 > 新建培训**。

### 2. 设置培训内容

| 字段 | 说明 |
|------|------|
| 培训主题 | 培训名称 |
| 培训类型 | 技能/制度/安全 |
| 培训方式 | 线下/视频/文档 |
| 参与人员 | 全员或指定 |
| 培训时间 | 日期和时长 |

### 3. 发布培训

员工会收到培训通知。

</Steps>

### 培训追踪

跟踪培训完成情况：
- 参与状态
- 完成进度
- 考核结果

### 技能档案

记录员工的技能和资质：

| 类型 | 说明 |
|------|------|
| 专业技能 | 美甲、美睫等级 |
| 产品知识 | 熟悉的产品线 |
| 资质证书 | 持有的专业证书 |
| 培训记录 | 完成的培训课程 |

## 考勤管理

### 考勤规则

设置门店的考勤规则：

| 规则 | 示例 |
|------|------|
| 迟到标准 | 超过班次开始5分钟 |
| 早退标准 | 提前班次结束10分钟 |
| 加班认定 | 超过排班时间30分钟 |

### 考勤记录

查看员工考勤详情：
- 每日打卡记录
- 迟到/早退统计
- 缺勤记录
- 请假记录

### 考勤异常

处理考勤异常：

<Steps>

### 1. 查看异常

进入 **团队 > 考勤 > 异常记录**。

### 2. 处理异常

- **补打卡** - 员工忘记打卡
- **标记为正常** - 特殊情况认定
- **维持异常** - 确认为违规

### 3. 记录说明

填写处理原因。

</Steps>

## 常见问题

<FAQ>
  <FAQItem question="如何查看员工的历史绩效？">
    进入员工详情页，点击「绩效」标签，可以查看该员工的历史绩效数据，支持按月份或自定义时间范围筛选。
  </FAQItem>
  <FAQItem question="绩效目标没达成有什么影响？">
    系统只做数据记录和提醒，不会自动产生惩罚。如何处理未达成目标由管理制度决定。
  </FAQItem>
  <FAQItem question="员工可以看到其他人的业绩吗？">
    默认员工只能看到自己的业绩。是否公开排名可以在系统设置中配置。
  </FAQItem>
  <FAQItem question="如何激励业绩好的员工？">
    可以通过系统发布表扬公告，或在绩效排名中给予荣誉标识。物质激励需要线下处理。
  </FAQItem>
</FAQ>
