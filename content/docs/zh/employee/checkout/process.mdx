# 结账流程

为客人完成结账的基本流程，包括开始结账、查看订单和应用优惠。

## 适用角色

- 前台 (Receptionist)
- 员工 (Employee)
- 经理 (Manager)

## 前置条件

- 已登录员工账号
- 拥有 `payments:process` 权限
- 服务已完成

## 开始结账

### 从预约进入

服务完成后：

<Steps>

### 1. 找到已完成的预约

在预约列表或看板中找到状态为「已完成」的预约。

### 2. 点击结账

点击 **结账** 按钮进入结账页面。

</Steps>

### 快速结账

在结账/收银页面：

1. 点击 **新建订单**
2. 搜索或选择客人
3. 添加服务项目
4. 进行结账

## 结账页面

### 订单信息

显示待结账内容：

| 项目 | 说明 |
|------|------|
| 服务项目 | 消费的服务列表 |
| 产品 | 购买的产品（如有） |
| 单价 | 每项价格 |
| 数量 | 项目数量 |
| 小计 | 项目总价 |

### 价格计算

| 项目 | 说明 |
|------|------|
| 服务小计 | 所有服务的总价 |
| 产品小计 | 产品销售额 |
| 折扣 | 优惠/折扣金额 |
| 税费 | 适用税金 |
| **应付金额** | 客人需支付的总额 |

## 应用优惠

### 优惠券

<Steps>

### 1. 点击添加优惠

在结账页面点击 **优惠券**。

### 2. 输入优惠券码

输入客人提供的优惠券码，或从客人账户选择可用优惠券。

### 3. 确认应用

系统自动计算折扣金额，更新应付金额。

</Steps>

### 会员折扣

如客人是会员：
- 系统自动识别会员等级
- 自动应用对应折扣
- 显示折扣明细

### 手动折扣

需要经理权限：

1. 点击 **手动折扣**
2. 选择折扣类型（百分比/固定金额）
3. 输入折扣值
4. 填写折扣原因
5. 确认应用

<Callout type="warning">
手动折扣会记录在日志中，用于审计。过度使用可能需要说明。
</Callout>

## 完成结账

### 打印小票

结账完成后：

1. 系统提示打印小票
2. 选择打印或跳过
3. 小票打印给客人

**小票内容：**
- 门店信息
- 服务明细
- 支付信息
- 消费日期时间
- 服务技师

### 发送电子收据

可以发送电子收据：

1. 输入客人邮箱
2. 发送电子收据
3. 或发送短信通知

### 结账成功

结账完成后：
- 预约状态更新为「已结账」
- 更新会员积分（如适用）
- 更新销售数据
