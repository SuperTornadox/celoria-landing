# 特殊情况

处理结账过程中的特殊情况，包括退款、挂账和价格调整。

## 退款

需要退款时：

<Steps>

### 1. 找到订单

在订单历史中找到需要退款的订单。

### 2. 点击退款

点击 **退款** 按钮。

### 3. 选择退款项目

选择需要退款的项目（可部分退款）。

### 4. 选择退款方式

- 原路退回
- 现金退款
- 退到会员余额

### 5. 确认退款

需要经理权限确认。

</Steps>

<Callout type="error">
退款操作会记录日志。频繁退款需要说明原因。
</Callout>

## 挂账

客人暂时无法支付：

1. 点击 **挂账**
2. 填写挂账原因
3. 设置还款期限
4. 确认挂账

<Callout type="warning">
挂账需要经理审批。请谨慎使用。
</Callout>

## 价格调整

发现价格错误时：

1. 点击对应项目
2. 选择 **调整价格**
3. 输入正确价格
4. 填写调整原因
5. 可能需要经理确认

## 常见问题

<FAQ>
  <FAQItem question="如何修改已完成的订单？">
    已完成的订单不能直接修改。如需调整，需要进行退款或补款操作，并记录原因。
  </FAQItem>
  <FAQItem question="小票打印机不工作怎么办？">
    检查打印机电源和连接。如果无法打印，可以发送电子收据给客人，或手写收据。
  </FAQItem>
</FAQ>
