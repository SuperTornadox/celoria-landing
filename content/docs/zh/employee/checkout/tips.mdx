# 小费处理

结账时添加和管理小费。

## 添加小费

结账时可以添加小费：

| 方式 | 说明 |
|------|------|
| 预设比例 | 15%、18%、20% 等快捷选项 |
| 自定义金额 | 手动输入小费金额 |
| 无小费 | 客人不给小费 |

## 小费支付

小费可以：
- 与服务费一起支付
- 单独现金支付

## 小费分配

小费分配给服务技师，具体规则在系统设置中配置。
