# 支付方式

支持多种支付方式，包括现金、信用卡、礼品卡、会员储值和混合支付。

## 现金支付

<Steps>

### 1. 选择现金

在支付方式中选择 **现金**。

### 2. 输入收到金额

输入客人支付的金额。

### 3. 计算找零

系统自动计算应找零金额。

### 4. 确认支付

确认后完成结账。

</Steps>

## 信用卡/借记卡

<Steps>

### 1. 选择刷卡

在支付方式中选择 **信用卡** 或 **借记卡**。

### 2. 处理刷卡

使用 POS 机刷卡：
- 插卡或刷卡
- 客人输入密码（如需要）
- 等待授权

### 3. 确认成功

POS 机返回成功后，在系统中确认。

</Steps>

## 礼品卡支付

<Steps>

### 1. 选择礼品卡

在支付方式中选择 **礼品卡**。

### 2. 输入卡号

输入礼品卡号或扫描卡片。

### 3. 确认余额

系统显示卡片余额。

### 4. 确认使用

- 余额充足：全额使用礼品卡
- 余额不足：使用全部余额，剩余部分选择其他支付方式

</Steps>

## 会员储值

如客人有储值余额：

1. 选择 **储值余额** 支付
2. 确认使用金额
3. 确认支付
4. 余额自动扣减

## 混合支付

一个订单可以使用多种支付方式：

<Steps>

### 1. 添加第一种支付方式

例如：礼品卡支付 $50

### 2. 添加第二种支付方式

例如：现金支付剩余 $30

### 3. 确认总额

确保各支付方式金额之和等于应付金额。

### 4. 完成结账

</Steps>

## 常见问题

<FAQ>
  <FAQItem question="刷卡失败怎么办？">
    首先检查网络连接和 POS 机状态。如果设备正常，请客人尝试另一张卡或使用其他支付方式。如仍无法解决，联系技术支持。
  </FAQItem>
  <FAQItem question="客人没带钱也没带卡怎么办？">
    可以使用挂账功能（需经理审批），或让客人通过手机支付（如支持）。如都不行，记录客人信息后让其稍后支付。
  </FAQItem>
</FAQ>
