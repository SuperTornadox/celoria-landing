# 预约管理

本文介绍员工如何查看和管理预约，包括查看分配的预约、开始服务、更新状态等操作。

## 适用角色

- 所有员工 (Employee)
- 前台 (Receptionist)
- 经理 (Manager)

## 前置条件

- 已登录员工账号
- 拥有 `appointments:view` 权限

## 查看我的预约

### 今日预约

登录后在首页或预约页面查看今日分配给自己的预约。

**显示信息：**
- 预约时间
- 客人姓名
- 服务项目
- 预约状态
- 客人备注（如有）

### 预约看板

进入 **预约 > 看板** 查看所有预约的可视化排布。

**看板视图：**
- 按时间轴显示
- 按员工分列
- 不同颜色表示状态
- 支持拖拽调整

### 预约列表

进入 **预约 > 列表** 查看表格形式的预约。

支持筛选：
- 按日期
- 按状态
- 按客人
- 按员工

## 预约状态流程

预约有以下状态：

```
待确认 → 已确认 → 已签到 → 服务中 → 已完成 → 已结账
                ↓
              取消/爽约
```

| 状态 | 说明 | 操作者 |
|------|------|--------|
| 待确认 | 新创建的预约 | 系统/前台 |
| 已确认 | 客人已确认 | 前台/系统 |
| 已签到 | 客人已到店 | 前台 |
| 服务中 | 正在提供服务 | 技师 |
| 已完成 | 服务完成 | 技师 |
| 已结账 | 已完成支付 | 前台/技师 |
| 已取消 | 预约被取消 | 客人/前台 |
| 爽约 | 客人未到 | 前台 |

## 服务操作

### 开始服务

当客人准备好开始服务时：

<Steps>

### 1. 找到预约

在今日预约或看板中找到该预约。

### 2. 确认客人

确认客人身份和服务需求。

### 3. 点击开始服务

点击 **开始服务** 按钮。

系统会：
- 记录服务开始时间
- 更新预约状态为「服务中」
- 开始计时

</Steps>

### 服务中操作

服务过程中可以：

**添加服务项目**
如客人需要增加服务：
1. 点击预约详情
2. 点击 **添加服务**
3. 选择额外服务
4. 确认添加

**记录备注**
记录服务中的特殊情况：
1. 点击预约详情
2. 在备注栏填写
3. 保存

**查看客人历史**
了解客人偏好：
- 历史服务记录
- 喜好备注
- 过敏信息

### 完成服务

服务完成后：

<Steps>

### 1. 点击完成服务

在预约详情中点击 **完成服务**。

### 2. 确认服务内容

核对最终服务项目：
- 原预约服务
- 新增服务
- 使用的产品

### 3. 添加备注（可选）

记录服务结果或客人反馈。

### 4. 确认完成

点击确认，预约状态变为「已完成」。

</Steps>

系统会：
- 记录服务结束时间
- 计算服务时长
- 准备结账信息

## 预约详情

点击预约可查看详细信息：

### 基本信息

| 信息 | 说明 |
|------|------|
| 预约编号 | 唯一预约标识 |
| 预约时间 | 预定的服务时间 |
| 创建时间 | 预约创建时间 |
| 创建方式 | 在线/电话/Walk-in |

### 客人信息

| 信息 | 说明 |
|------|------|
| 客人姓名 | 客人名称 |
| 联系电话 | 客人手机号 |
| 会员等级 | VIP/普通等 |
| 历史备注 | 客人的特殊需求 |

### 服务信息

| 信息 | 说明 |
|------|------|
| 服务项目 | 预约的服务列表 |
| 预计时长 | 服务预估时间 |
| 预计金额 | 服务预估费用 |
| 分配技师 | 负责的技师 |

## 修改预约

### 修改时间

如客人需要改期：

<Steps>

### 1. 打开预约详情

### 2. 点击修改时间

### 3. 选择新的日期和时间

查看可用时段。

### 4. 确认修改

系统会发送变更通知给客人。

</Steps>

### 修改技师

更换服务技师：

1. 打开预约详情
2. 点击技师名称
3. 选择其他可用技师
4. 确认修改

<Callout type="warning">
修改技师前建议与客人沟通，特别是指定技师的预约。
</Callout>

### 修改服务

调整服务项目：

1. 打开预约详情
2. 点击 **修改服务**
3. 添加或移除服务项目
4. 保存修改

## 取消预约

### 客人主动取消

收到客人取消请求：

1. 找到对应预约
2. 点击 **取消预约**
3. 选择取消原因
4. 确认取消

系统会：
- 释放时间段
- 发送取消确认给客人
- 处理预付款（如有）

### 门店取消

特殊情况需要门店取消：

1. 找到对应预约
2. 点击 **取消预约**
3. 选择门店原因
4. 添加说明
5. 确认取消

<Callout type="error">
门店主动取消应提前通知客人并表示歉意，必要时提供补偿。
</Callout>

## 客人备注

### 查看备注

预约中可能包含客人的特殊需求：

- 过敏信息（如对某产品过敏）
- 偏好（如喜欢安静）
- 特殊要求（如赶时间）

### 添加服务备注

服务过程中添加的备注：

- 使用的颜色/款式
- 客人满意度
- 下次建议

这些备注会保存到客人档案，方便下次服务参考。

## 常见问题

<FAQ>
  <FAQItem question="预约时间快到了但客人还没到怎么办？">
    可以主动联系客人确认是否能准时到达。如超过预约时间 15 分钟仍未到达且无法联系，可以考虑标记为爽约。
  </FAQItem>
  <FAQItem question="服务时间超出预约时长怎么办？">
    服务质量优先，适当超时是可以接受的。但如果影响下一位客人，需要提前告知并道歉。
  </FAQItem>
  <FAQItem question="客人对服务不满意要求重做怎么办？">
    先了解不满意的原因，如确实是服务问题，应积极处理。重做或补救服务需要在系统中记录，并可能需要经理审批。
  </FAQItem>
  <FAQItem question="如何查看客人的历史偏好？">
    点击客人姓名进入客人详情页，可以查看历史服务记录、备注和偏好设置。
  </FAQItem>
</FAQ>
