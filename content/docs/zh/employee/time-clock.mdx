# 打卡操作

本文介绍员工如何使用 Celoria 进行上下班打卡和查看考勤记录。

<Video id="V12" title="打卡操作视频教程" duration="2-3 分钟" />

## 适用角色

- 所有员工 (Employee)

## 前置条件

- 已登录员工账号
- 已分配门店和排班

## 打卡方式

Celoria 支持多种打卡方式。

### Web 端打卡

<Steps>

### 1. 登录系统

使用员工账号登录 Celoria。

### 2. 进入打卡页面

点击顶部导航栏的 **打卡** 按钮，或进入 **员工 > 打卡**。

### 3. 确认信息

检查显示的信息：
- 当前时间
- 所在门店
- 今日排班

### 4. 点击打卡

- **上班打卡** - 点击 **上班** 按钮
- **下班打卡** - 点击 **下班** 按钮

### 5. 打卡成功

系统显示打卡成功信息，记录打卡时间。

</Steps>

### 移动端打卡

员工可以在手机上使用 Celoria App 打卡：

1. 打开 Celoria App
2. 登录员工账号
3. 点击首页的 **打卡** 按钮
4. 确认打卡

<Callout type="info">
移动端打卡可能需要开启定位权限，以验证是否在门店范围内。
</Callout>

### PIN 码打卡（店内设备）

部分门店可能配置了公共打卡终端：

1. 在店内打卡终端上操作
2. 输入个人 PIN 码
3. 选择上班/下班
4. 完成打卡

## 打卡状态

### 正常打卡

在排班时间前后的容差范围内打卡：
- 上班打卡容差：排班开始前后 5 分钟（可配置）
- 下班打卡容差：排班结束前后 10 分钟（可配置）

### 迟到

上班打卡时间超过排班开始时间+容差。

**示例：** 排班 9:00，容差 5 分钟，9:06 打卡 = 迟到

### 早退

下班打卡时间早于排班结束时间-容差。

**示例：** 排班 17:00 结束，容差 10 分钟，16:45 打卡 = 早退

### 缺卡

排班日未打卡或只打了一次卡。

### 加班

超出排班时间的工作时长。

## 查看打卡记录

### 今日记录

在打卡页面可以看到：
- 今日上班打卡时间
- 今日下班打卡时间
- 工作时长
- 打卡状态（正常/迟到/早退）

### 历史记录

<Steps>

### 1. 进入考勤记录

点击 **员工 > 考勤记录**。

### 2. 选择日期范围

选择要查看的时间范围：
- 本周
- 本月
- 自定义日期

### 3. 查看详情

查看每天的打卡记录和状态。

</Steps>

### 记录导出

可导出考勤记录用于：
- 工资核算
- 个人存档

## 补打卡

如果忘记打卡，可以申请补打卡。

<Steps>

### 1. 提交补卡申请

进入 **考勤记录**，找到缺卡的日期，点击 **申请补卡**。

### 2. 填写信息

| 字段 | 说明 |
|------|------|
| 补卡日期 | 需要补卡的日期 |
| 补卡类型 | 上班卡/下班卡 |
| 补卡时间 | 实际应打卡的时间 |
| 补卡原因 | 说明忘记打卡的原因 |

### 3. 提交审批

提交后等待经理审批。

### 4. 审批结果

- **通过** - 补卡生效，考勤更新
- **拒绝** - 维持缺卡状态

</Steps>

<Callout type="warning">
补卡申请需要经理审批。建议尽量在当天或次日提交，超过一定时间可能无法补卡。
</Callout>

## 工时统计

### 本周工时

查看本周累计工作时长：
- 总工时
- 正常工时
- 加班工时

### 月度汇总

| 统计项 | 说明 |
|--------|------|
| 应出勤天数 | 本月排班天数 |
| 实际出勤 | 已打卡天数 |
| 迟到次数 | 迟到的天数 |
| 早退次数 | 早退的天数 |
| 缺勤天数 | 未打卡天数 |
| 总工时 | 累计工作小时数 |

## 常见问题

<FAQ>
  <FAQItem question="打卡按钮点不了怎么办？">
    检查是否在排班范围内。如果不在排班时间，打卡按钮可能被禁用。如果确实有排班但无法打卡，请联系经理。
  </FAQItem>
  <FAQItem question="打卡地点有要求吗？">
    部分门店启用了地理围栏功能，需要在门店一定范围内才能打卡。如果不在范围内会收到提示。
  </FAQItem>
  <FAQItem question="忘记打下班卡怎么办？">
    提交补卡申请，说明原因后等待经理审批。建议设置下班提醒避免忘记。
  </FAQItem>
  <FAQItem question="打卡记录可以修改吗？">
    员工无法直接修改打卡记录。如有错误，需要通过补卡申请或联系经理处理。
  </FAQItem>
</FAQ>
