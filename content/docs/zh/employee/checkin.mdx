# 客人签到

本文介绍如何使用 Celoria 处理客人到店签到和接待流程。

## 适用角色

- 前台 (Receptionist)
- 员工 (Employee)
- 经理 (Manager)

## 前置条件

- 已登录员工账号
- 拥有 `appointments:view` 权限

## 签到方式

### 有预约客人

客人提前预约后到店。

<Steps>

### 1. 确认客人信息

询问客人姓名或预约信息。

### 2. 在系统中查找

在 **预约列表** 或 **今日预约** 中找到该预约。

查找方式：
- 按客人姓名搜索
- 按预约时间筛选
- 按手机号查找

### 3. 确认预约详情

核对预约信息：
- 预约时间
- 预约服务
- 指定技师（如有）

### 4. 点击签到

在预约详情中点击 **客人已到** 或 **签到** 按钮。

### 5. 通知技师

系统会自动通知分配的技师，或手动通知。

</Steps>

### Walk-in 客人

没有预约直接到店的客人。

<Steps>

### 1. 了解需求

询问客人需要的服务。

### 2. 检查可用性

查看当前可用的技师和时间：
- 打开 **预约看板**
- 查看空闲时段
- 确认等待时间

### 3. 创建临时预约

如果可以接待：
1. 点击 **新建预约**
2. 填写客人信息
3. 选择服务项目
4. 分配技师
5. 保存并签到

### 4. 告知等待时间

如需等待，告知客人：
- 预计等待时间
- 可以先做登记
- 提供等候区引导

</Steps>

<Callout type="info">
Walk-in 客人建议创建客户档案，方便下次预约和积累会员数据。
</Callout>

### 自助签到

部分门店可能配置自助签到设备。

**客人操作流程：**
1. 在自助终端输入手机号或扫描预约码
2. 确认预约信息
3. 完成签到
4. 系统自动通知前台和技师

## 签到页面

### 今日预约列表

显示今天所有预约：

| 列 | 说明 |
|------|------|
| 时间 | 预约时间 |
| 客人 | 客人姓名 |
| 服务 | 预约的服务 |
| 技师 | 分配的技师 |
| 状态 | 待签到/已签到/服务中/已完成 |

### 预约状态颜色

| 颜色 | 状态 | 说明 |
|------|------|------|
| 白色 | 待签到 | 等待客人到达 |
| 蓝色 | 已签到 | 客人已到，等待服务 |
| 绿色 | 服务中 | 正在进行服务 |
| 灰色 | 已完成 | 服务完成 |
| 红色 | 迟到/爽约 | 超时未到或取消 |

### 快捷操作

在预约列表中可直接：
- **签到** - 标记客人到达
- **开始** - 开始服务
- **改期** - 修改预约时间
- **取消** - 取消预约

## 等候管理

### 等候队列

当客人签到后等待服务时，进入等候队列。

查看等候队列：
- 当前等候人数
- 每位客人的等待时长
- 预计服务开始时间

### 叫号服务

如门店配置了叫号系统：
1. 客人签到后获得号码
2. 在等候区等待
3. 技师空闲时系统叫号
4. 显示屏或短信通知客人

### 优先级调整

特殊情况可调整等候顺序：
- VIP 客户优先
- 预约客户优先
- 特殊需求处理

<Callout type="warning">
调整等候顺序时应妥善处理其他客人的期望，避免引起不满。
</Callout>

## 签到后流程

### 引导客人

签到完成后：
1. 引导客人到等候区或工位
2. 提供饮品（如有）
3. 介绍预计等待时间
4. 提醒可用的设施（WiFi、杂志等）

### 通知技师

系统自动通知或手动告知：
- 技师姓名
- 客人信息
- 服务项目
- 特殊备注

### 准备工位

确认服务工位准备就绪：
- 工位清洁
- 工具准备
- 产品准备

## 异常处理

### 客人迟到

**迟到 15 分钟内**
- 尽量按原计划服务
- 可能需要缩短服务时间
- 告知客人可能的影响

**迟到超过 15 分钟**
- 联系客人确认是否能来
- 评估是否影响后续预约
- 考虑改期或取消

### 客人爽约

超过预约时间 30 分钟未到且无法联系：

1. 标记为 **爽约**
2. 释放技师时间
3. 记录爽约次数
4. 按政策处理（如收取违约费）

### 客人提前到

- 如技师空闲，可提前开始
- 如需等待，告知预计时间
- 提供舒适的等候环境

## 常见问题

<FAQ>
  <FAQItem question="客人忘记预约时间怎么办？">
    通过手机号查询客人的预约记录，确认预约详情。如果没有预约记录，可以当作 Walk-in 客人处理。
  </FAQItem>
  <FAQItem question="预约的技师不在怎么办？">
    先向客人说明情况并道歉，询问是否接受其他技师服务，或者改期到指定技师有空的时间。
  </FAQItem>
  <FAQItem question="多个客人同时到怎么办？">
    按预约时间顺序签到，先到先服务。如果都是 Walk-in，按到达顺序排队。
  </FAQItem>
  <FAQItem question="签到后客人临时离开怎么办？">
    了解离开原因和预计返回时间。如果时间较长，可能需要调整排队顺序或重新安排。
  </FAQItem>
</FAQ>
