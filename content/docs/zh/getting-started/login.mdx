# 登录与导航

本文介绍如何登录 Celoria 系统并熟悉界面布局。

<Video id="V01" title="登录与界面导航" duration="2-3 分钟" />

## 适用角色

- 管理员 (Admin)
- 经理 (Manager)
- 员工 (Employee)

## 登录系统

<Steps>

### 1. 打开登录页面

在浏览器中访问您的 Celoria 系统地址，例如 `https://your-salon.celoria.ai`。

### 2. 输入账号信息

在登录页面输入您的：
- **邮箱地址** - 管理员分配给您的工作邮箱
- **密码** - 首次登录使用初始密码，建议立即修改

### 3. 点击登录

点击 **登录** 按钮进入系统。

</Steps>

<Callout type="tip">
如果忘记密码，点击"忘记密码"链接，系统会发送重置邮件到您的邮箱。
</Callout>

## 界面布局

登录后，您将看到主界面，主要分为以下几个区域：

### 顶部导航栏

- **Logo** - 点击返回首页
- **门店选择** - 切换不同门店（多店铺适用）
- **通知图标** - 查看系统通知
- **用户菜单** - 个人设置、退出登录

### 侧边导航菜单

根据您的角色权限，侧边栏会显示不同的菜单项：

| 菜单项 | 说明 | 权限要求 |
|--------|------|----------|
| 仪表盘 | 数据概览 | 所有角色 |
| 预约 | 预约管理 | 所有角色 |
| 客户 | 客户管理 | 员工及以上 |
| 员工 | 员工管理 | 经理及以上 |
| 报表 | 数据报表 | 经理及以上 |
| 设置 | 系统设置 | 管理员 |

### 主内容区

显示当前页面的具体内容，如预约看板、客户列表等。

## 切换门店

如果您管理多个门店，可以通过顶部导航栏的门店选择器切换：

1. 点击顶部的门店名称
2. 在下拉菜单中选择目标门店
3. 页面会自动刷新显示该门店数据

<Callout type="warning">
切换门店后，所有数据都会变为目标门店的数据，请注意确认当前门店。
</Callout>

## 常见问题

<FAQ>
  <FAQItem question="登录时提示密码错误怎么办？">
    请先确认邮箱地址输入正确。如果仍然无法登录，点击"忘记密码"重置密码，或联系管理员。
  </FAQItem>
  <FAQItem question="为什么我看不到某些菜单？">
    菜单显示取决于您的角色权限。如需访问更多功能，请联系管理员调整权限。
  </FAQItem>
  <FAQItem question="如何修改登录密码？">
    登录后，点击右上角用户菜单 → 个人设置 → 修改密码。
  </FAQItem>
</FAQ>
