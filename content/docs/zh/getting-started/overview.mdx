# 产品概览

Celoria 是一款专为美甲沙龙设计的智能管理系统，帮助您高效管理门店运营的方方面面。

## 核心功能

Celoria 提供以下核心功能模块：

### 预约管理

- **预约看板** - 直观的日历视图，一目了然查看所有预约
- **在线预约** - 客人可通过网页或手机自助预约
- **智能排程** - 自动检测时间冲突，优化员工排班

### 客户管理

- **客户档案** - 记录客户偏好、历史消费、会员等级
- **标签分群** - 智能标签系统，精准营销
- **会员体系** - 灵活的积分和等级制度

### 员工管理

- **排班系统** - 周模板、自动排班、调班管理
- **打卡考勤** - PIN码打卡，时卡自动统计
- **业绩统计** - 实时查看员工服务和销售数据

### 财务管理

- **收银结账** - 支持多种支付方式、拆分支付
- **日结关账** - 5步流程，清晰对账
- **报表分析** - 销售、利润、工时等多维度报表

### 营销工具

- **邮件/短信** - 模板化消息，自动化触达
- **促销活动** - 折扣码、满减、会员专享
- **礼品卡** - 在线购买、实体卡管理

## 适用场景

Celoria 适用于：

- 单店美甲沙龙
- 连锁美甲品牌
- 美容美发综合店
- SPA 水疗中心
