# 销售报表

查看营收相关数据，分析销售趋势。

## 适用角色

- 超级管理员 (Super Admin)
- 管理员 (Admin)
- 经理 (Manager) - 门店级报表

## 前置条件

- 已登录管理员或经理账号
- 拥有 `reports:view` 权限

## 主要指标

- 总营收
- 订单数量
- 平均客单价
- 支付方式分布

## 筛选维度

- 日期范围
- 门店
- 服务类别
- 员工

<Steps>

### 查看销售报表

1. 进入 **报表 > 销售报表**
2. 选择日期范围（今日/本周/本月/自定义）
3. 选择门店（管理员可看所有门店）
4. 查看数据图表和明细

</Steps>

## 销售趋势图

显示选定时间范围内的销售趋势：

- **日视图** - 每小时销售额
- **周视图** - 每天销售额
- **月视图** - 每周销售额

## 销售明细

详细的交易记录列表：

| 列 | 说明 |
|------|------|
| 日期时间 | 交易时间 |
| 订单号 | 唯一订单编号 |
| 客户 | 客户姓名 |
| 服务 | 消费的服务项目 |
| 员工 | 服务技师 |
| 金额 | 消费金额 |
| 支付方式 | 现金/卡/礼品卡等 |
