# 员工绩效报表

查看员工的业绩排名、工时统计和服务效率。

## 业绩排名

按服务营收排名员工：

| 指标 | 说明 |
|------|------|
| 服务收入 | 员工完成服务的总金额 |
| 服务次数 | 完成的服务数量 |
| 平均客单价 | 每次服务的平均金额 |
| 小费收入 | 收到的小费总额 |

## 工时统计

员工的工作时间统计：

- 出勤天数
- 总工作时长
- 平均每日工时
- 迟到/早退次数

## 服务效率

- 平均服务时长
- 服务完成率
- 客户评价得分

<Callout type="info">
员工可以查看自己的绩效报表。经理可以查看本店所有员工。管理员可以查看全部。
</Callout>

## 数据权限

不同角色看到的数据范围：

| 角色 | 数据范围 |
|------|----------|
| 超级管理员 | 所有门店所有数据 |
| 管理员 | 所有门店所有数据 |
| 经理 | 所属门店数据 |
| 员工 | 仅自己的数据 |
