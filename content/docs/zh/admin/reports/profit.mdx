# 利润报表

分析收入与支出，了解门店盈利状况。

## 收入与支出

| 项目 | 说明 |
|------|------|
| 服务收入 | 来自服务的营收 |
| 产品收入 | 产品销售收入 |
| 礼品卡收入 | 礼品卡销售 |
| 员工成本 | 工资和提成支出 |
| 产品成本 | 消耗品成本 |
| 其他支出 | 其他运营成本 |
| **净利润** | 收入减去支出 |
