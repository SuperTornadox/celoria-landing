# 导出报表

所有报表支持导出为多种格式。

## 导出格式

| 格式 | 适用场景 |
|------|----------|
| PDF | 打印或存档 |
| Excel | 进一步分析处理 |
| CSV | 导入其他系统 |

## 导出步骤

1. 设置报表筛选条件
2. 点击 **导出** 按钮
3. 选择导出格式
4. 下载文件

<Callout type="warning">
导出大量数据可能需要一些时间，请耐心等待。
</Callout>

## 常见问题

<FAQ>
  <FAQItem question="可以查看历史数据吗？">
    可以。系统保留所有历史数据，在日期筛选中选择对应日期范围即可查看。
  </FAQItem>
  <FAQItem question="如何打印报表？">
    导出PDF格式后打印，或直接使用浏览器的打印功能（Ctrl+P / Cmd+P）。
  </FAQItem>
</FAQ>
