# 邮件营销

创建和管理邮件模板，支持富文本格式和可视化编辑。

## 邮件模板

邮件模板支持富文本格式和图片。

## 创建邮件模板

<Steps>

### 1. 进入邮件模板

点击 **营销 > 邮件模板**。

### 2. 新建模板

选择从空白创建或使用预设模板。

### 3. 编辑内容

使用可视化编辑器：
- 添加标题和正文
- 插入图片和按钮
- 设置样式和颜色
- 使用变量标签

### 4. 设置邮件信息

- 邮件主题
- 发件人名称
- 回复地址

### 5. 预览和测试

发送测试邮件到自己邮箱确认效果。

</Steps>

## 邮件类型

| 类型 | 用途 | 触发时机 |
|------|------|----------|
| 欢迎邮件 | 新客户注册 | 客户首次创建账号 |
| 预约确认 | 确认预约信息 | 预约创建后 |
| 收据邮件 | 消费明细 | 结账完成后 |
| 促销邮件 | 活动推广 | 手动或定时发送 |
| 回访邮件 | 服务满意度 | 服务完成后 |
