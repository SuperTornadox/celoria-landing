# 自动化营销

设置条件自动执行营销动作，让营销更高效。

## 触发器规则

<Steps>

### 1. 创建触发器

点击 **营销 > 自动化 > 新建触发器**。

### 2. 设置触发条件

**时间触发**
- 客户生日前N天
- 上次到店后N天
- 会员到期前N天

**事件触发**
- 新客户注册
- 首次预约
- 消费满N元

### 3. 设置动作

触发后执行的动作：
- 发送短信
- 发送邮件
- 赠送优惠券
- 添加标签

### 4. 测试和启用

测试触发器正常工作后启用。

</Steps>

## 常用自动化场景

**生日营销**
```
触发：客户生日前3天
动作：发送生日祝福短信 + 赠送生日优惠券
```

**流失预警**
```
触发：30天未到店
动作：发送回访短信 + 发送专属优惠
```

**新客转化**
```
触发：新客户首次到店后7天
动作：发送满意度调查 + 赠送二次到店优惠
```
