# 促销活动

管理各类促销活动，包括折扣、满减、买赠和套餐价等。

## 适用角色

- 超级管理员 (Super Admin)
- 管理员 (Admin)

## 前置条件

- 已登录管理员账号
- 拥有 `marketing:manage` 权限

## 创建促销

<Steps>

### 1. 进入促销管理

点击 **营销 > 促销活动**。

### 2. 点击新建促销

在页面右上角点击 **新建促销**。

### 3. 选择促销类型

| 类型 | 说明 | 示例 |
|------|------|------|
| 折扣 | 按百分比折扣 | 全场8折 |
| 满减 | 满足金额减免 | 满$100减$20 |
| 买赠 | 购买指定服务送礼 | 做美甲送护手霜 |
| 套餐价 | 多服务组合优惠 | 美甲+美足$80 |

### 4. 设置促销详情

**基本信息**
- 促销名称
- 促销描述（客户可见）
- 生效日期范围

**优惠规则**
- 折扣比例或减免金额
- 适用服务范围
- 适用门店

**使用条件**
- 最低消费金额
- 每人限用次数
- 可叠加其他优惠

### 5. 保存并发布

保存后选择立即发布或定时发布。

</Steps>

## 促销状态

| 状态 | 说明 |
|------|------|
| 草稿 | 未发布，可编辑 |
| 待生效 | 已发布，等待开始日期 |
| 进行中 | 正在进行的促销 |
| 已结束 | 促销已过期 |
| 已停用 | 手动停用的促销 |

## 营销数据分析

### 活动效果

查看促销活动的：
- 参与人数
- 使用次数
- 带来营收
- ROI（投资回报率）

## 常见问题

<FAQ>
  <FAQItem question="如何停止正在进行的促销？">
    在促销详情页点击「停用」按钮。已使用的优惠不会撤回，但新客户将无法使用该促销。
  </FAQItem>
</FAQ>
