# 优惠券

创建和管理各类优惠券，支持手动发放、自动发放和客户自助领取。

## 创建优惠券

<Steps>

### 1. 进入优惠券管理

点击 **营销 > 优惠券**。

### 2. 新建优惠券

点击 **新建优惠券**。

### 3. 设置优惠券信息

| 字段 | 说明 |
|------|------|
| 名称 | 优惠券显示名称 |
| 类型 | 折扣券/代金券/免费服务 |
| 面值 | 折扣比例或金额 |
| 有效期 | 使用期限 |
| 使用条件 | 最低消费、适用服务 |

### 4. 设置发放方式

- **手动发放** - 为指定客户发放
- **自动发放** - 符合条件自动获得
- **领取链接** - 客户自助领取

### 5. 保存优惠券

</Steps>

## 优惠券使用

1. 客户结账时出示优惠券
2. 员工在结账页面输入优惠券码
3. 系统自动计算折扣
4. 确认使用后优惠券标记为已用

## 常见问题

<FAQ>
  <FAQItem question="优惠券可以叠加使用吗？">
    默认不可叠加。如需允许叠加，在创建优惠券时勾选「可与其他优惠同时使用」选项。
  </FAQItem>
</FAQ>
