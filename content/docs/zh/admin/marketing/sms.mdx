# 短信营销

管理短信模板和短信发送，包括预设模板、自定义模板和自动发送。

## 短信模板

系统提供预设的短信模板：

- **预约提醒** - 预约前自动发送
- **生日祝福** - 客户生日当天发送
- **回访邀请** - 一段时间未到店提醒
- **促销通知** - 新活动通知

## 创建自定义模板

<Steps>

### 1. 进入短信模板

点击 **营销 > 短信模板**。

### 2. 新建模板

点击 **新建模板** 按钮。

### 3. 编辑内容

使用变量标签插入动态内容：

| 变量 | 说明 |
|------|------|
| `{{guest.name}}` | 客户姓名 |
| `{{guest.firstName}}` | 客户名字 |
| `{{appointment.date}}` | 预约日期 |
| `{{appointment.time}}` | 预约时间 |
| `{{store.name}}` | 门店名称 |
| `{{store.phone}}` | 门店电话 |

**示例模板：**
```
Hi {{guest.firstName}}, 您在{{store.name}}的预约提醒：
日期：{{appointment.date}}
时间：{{appointment.time}}
如需取消或改期，请致电 {{store.phone}}
```

### 4. 预览和保存

预览确认无误后保存模板。

</Steps>

<Callout type="warning">
短信有字数限制（通常160字符），超出会按多条计费。请保持内容简洁。
</Callout>

## 发送短信

**手动发送**
1. 选择目标客户（可按条件筛选）
2. 选择短信模板
3. 预览内容
4. 确认发送

**自动发送**
配置触发条件后系统自动发送：
- 预约确认后
- 预约前24小时
- 客户生日
- N天未到店

## 常见问题

<FAQ>
  <FAQItem question="短信发送失败怎么办？">
    检查客户手机号格式是否正确。如号码正确但仍失败，可能是运营商问题，建议稍后重试或联系客服。
  </FAQItem>
  <FAQItem question="营销短信会被当作垃圾短信吗？">
    系统使用正规短信通道，通常不会被拦截。建议在合适的时间发送，避免打扰客户。
  </FAQItem>
</FAQ>
