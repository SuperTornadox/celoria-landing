# 门店设置

本文介绍如何在 Celoria 中创建和配置门店。

<Video id="V02" title="门店设置视频教程" duration="3-4 分钟" />

## 适用角色

- 超级管理员 (Super Admin)
- 管理员 (Admin)

## 前置条件

- 已登录管理员账号
- 拥有 `stores:create` 或 `stores:update` 权限

## 创建新门店

<Steps>

### 1. 进入门店管理

点击侧边栏 **门店** 菜单，进入门店列表页面。

### 2. 点击添加门店

点击页面右上角的 **添加门店** 按钮。

### 3. 填写基本信息

在弹出的表单中填写以下信息：

| 字段 | 说明 | 必填 |
|------|------|------|
| 门店名称 | 门店的显示名称 | 是 |
| 门店代码 | 唯一标识，创建后不可修改 | 是 |
| 联系电话 | 门店联系电话 | 是 |
| 邮箱 | 门店邮箱地址 | 否 |
| 地址 | 门店详细地址 | 是 |

<Callout type="warning">
门店代码创建后不可修改，请谨慎填写。建议使用简短有意义的代码，如 "NYC01"、"LA02"。
</Callout>

### 4. 设置营业时间

配置每周的营业时间：

1. 选择工作日（可多选）
2. 设置开始时间和结束时间
3. 如有午休时间，可添加休息时段

### 5. 保存门店

确认信息无误后，点击 **保存** 按钮完成创建。

</Steps>

## 编辑门店信息

1. 在门店列表中，点击目标门店的 **编辑** 按钮
2. 修改需要更新的信息
3. 点击 **保存** 确认修改

<Callout type="info">
门店代码无法修改。如需更改代码，需要创建新门店并迁移数据。
</Callout>

## 管理门店状态

门店有以下状态：

| 状态 | 说明 | 影响 |
|------|------|------|
| 营业中 | 正常运营状态 | 可接受预约 |
| 暂停营业 | 临时关闭 | 不接受新预约，现有预约保留 |
| 已关闭 | 永久关闭 | 不显示在门店列表 |

### 更改门店状态

1. 进入门店详情页
2. 点击 **门店状态** 下拉菜单
3. 选择新状态
4. 确认更改

<Callout type="error">
将门店设为"已关闭"状态后，该门店的所有未完成预约将被取消。请确保提前通知相关客人。
</Callout>

## 配置门店设置

除基本信息外，还可以配置以下门店级别设置：

### 预约设置

- **提前预约时间** - 客人最早可提前多久预约
- **预约间隔** - 可选时间段的间隔（如15分钟、30分钟）
- **取消政策** - 客人可提前多久免费取消

### 通知设置

- **预约确认** - 是否发送预约确认短信/邮件
- **预约提醒** - 预约前多久发送提醒
- **营销通知** - 是否接收营销消息

### 支付设置

- **接受的支付方式** - 现金、信用卡、礼品卡等
- **小费选项** - 预设小费比例
- **税率** - 适用的税率

## 常见问题

<FAQ>
  <FAQItem question="如何修改门店营业时间？">
    进入门店详情页，点击"编辑"按钮，在"营业时间"区域进行修改。修改后记得保存。
  </FAQItem>
  <FAQItem question="可以删除门店吗？">
    已有预约记录或交易记录的门店不能删除，只能设为"已关闭"状态。这是为了保留历史数据的完整性。
  </FAQItem>
  <FAQItem question="如何为门店添加员工？">
    请参考 [员工管理](/docs/admin/employee-management) 文档，在添加或编辑员工时指定所属门店。
  </FAQItem>
  <FAQItem question="多个门店可以共享员工吗？">
    可以。一个员工可以分配到多个门店，但每个门店需要单独设置该员工的排班。
  </FAQItem>
</FAQ>
