# 员工管理

本文介绍如何在 Celoria 中管理员工，包括添加新员工、编辑信息、分配职位和门店等操作。

<Video id="V03" title="员工管理视频教程" duration="4-5 分钟" />

## 适用角色

- 超级管理员 (Super Admin)
- 管理员 (Admin)
- 经理 (Manager) - 仅限查看和编辑本店员工

## 前置条件

- 已登录管理员或经理账号
- 拥有 `employees:create` 或 `employees:update` 权限

## 员工列表

进入 **员工** 菜单查看所有员工列表。列表显示：

| 信息 | 说明 |
|------|------|
| 头像 | 员工照片 |
| 姓名 | 员工姓名 |
| 职位 | 当前职位 |
| 所属门店 | 分配的门店 |
| 状态 | 在职/离职 |
| 联系方式 | 电话和邮箱 |

### 筛选和搜索

- **搜索框** - 按姓名或电话搜索
- **门店筛选** - 按门店筛选员工
- **状态筛选** - 在职/离职/全部

## 添加新员工

<Steps>

### 1. 点击添加按钮

在员工列表页面右上角，点击 **添加员工** 按钮。

### 2. 填写基本信息

| 字段 | 说明 | 必填 |
|------|------|------|
| 姓名 | 员工全名 | 是 |
| 手机号 | 用于登录和接收通知 | 是 |
| 邮箱 | 工作邮箱 | 否 |
| 员工代码 | 唯一标识（如工号） | 否 |

### 3. 分配职位

从下拉菜单选择员工职位：

- 美甲师 (Nail Technician)
- 前台 (Receptionist)
- 经理 (Manager)
- 其他自定义职位

<Callout type="info">
职位决定了员工的基础权限。具体权限可在「权限管理」中调整。
</Callout>

### 4. 分配门店

选择员工工作的门店。一个员工可以分配到多个门店。

### 5. 设置登录凭证

- **临时密码** - 系统生成或手动设置
- **首次登录** - 员工首次登录需修改密码

### 6. 保存员工

确认信息后点击 **保存**，新员工即添加成功。

</Steps>

<Callout type="warning">
手机号是员工的登录账号，请确保填写正确。同一手机号只能注册一个员工账号。
</Callout>

## 编辑员工信息

1. 在员工列表中找到目标员工
2. 点击员工姓名或 **编辑** 按钮
3. 修改需要更新的信息
4. 点击 **保存** 确认

### 可编辑字段

- 基本信息（姓名、联系方式）
- 职位和角色
- 所属门店
- 账号状态

## 员工详情页

点击员工姓名进入详情页，可查看：

### 基本信息卡片

显示员工的个人信息、联系方式和职位。

### 服务技能

员工可提供的服务项目及技能等级。

### 排班信息

本周和下周的排班安排，链接到排班管理。

### 业绩统计

- 本月完成预约数
- 本月营业额
- 平均客单价
- 客户评价

## 员工状态管理

### 在职状态

正常工作状态，可接受预约分配。

### 请假状态

临时不可用，不接受新预约。

<Steps>

### 设置请假

1. 进入员工详情页
2. 点击 **设置请假**
3. 选择请假日期范围
4. 填写请假原因（可选）
5. 确认保存

</Steps>

### 离职处理

<Steps>

### 1. 进入员工详情页

找到需要办理离职的员工。

### 2. 点击离职按钮

在页面右上角点击 **办理离职**。

### 3. 选择离职日期

设置最后工作日。

### 4. 处理现有预约

系统会显示该员工未来的预约，选择处理方式：
- 转给其他员工
- 取消预约并通知客户

### 5. 确认离职

离职后员工账号将被禁用，历史数据保留。

</Steps>

<Callout type="error">
离职操作不可撤销。如需恢复，请联系超级管理员。
</Callout>

## 批量操作

### 批量导入

支持通过 Excel 文件批量导入员工：

1. 下载导入模板
2. 按模板格式填写员工信息
3. 上传文件并预览
4. 确认导入

### 批量调整门店

选择多个员工，一次性调整所属门店。

## 常见问题

<FAQ>
  <FAQItem question="员工忘记密码怎么办？">
    管理员可以在员工详情页点击「重置密码」，系统会生成新的临时密码。员工也可以在登录页面通过手机号自助找回密码。
  </FAQItem>
  <FAQItem question="如何给员工分配多个门店？">
    在编辑员工信息时，门店字段支持多选。选择多个门店后保存即可。员工登录后可以切换不同门店工作。
  </FAQItem>
  <FAQItem question="离职员工的数据会被删除吗？">
    不会。离职员工的账号会被禁用，但所有历史数据（预约记录、业绩等）都会保留，可用于报表统计。
  </FAQItem>
  <FAQItem question="员工可以自己修改个人信息吗？">
    员工可以修改部分个人信息（如头像、联系方式）。职位、门店等关键信息只能由管理员修改。
  </FAQItem>
</FAQ>
