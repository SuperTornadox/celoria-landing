# 积分系统

配置积分的获取和使用规则。

## 获取积分

| 方式 | 默认比例 | 说明 |
|------|----------|------|
| 消费 | $1=1分 | 按消费金额获得积分 |
| 首次注册 | 100分 | 新会员注册奖励 |
| 生日 | 双倍积分 | 生日当天消费双倍积分 |
| 推荐新客 | 200分 | 推荐新客户注册 |

## 使用积分

| 方式 | 比例 | 说明 |
|------|------|------|
| 抵扣消费 | 100分=$1 | 积分当钱花 |
| 兑换礼品 | 按礼品设置 | 兑换指定礼品 |
| 兑换服务 | 按服务设置 | 兑换免费服务 |

## 配置积分规则

<Steps>

### 1. 进入积分设置

点击 **会员 > 积分设置**。

### 2. 设置获取规则

配置消费积分比例、奖励积分等。

### 3. 设置使用规则

配置积分抵扣比例、使用限制等。

### 4. 设置有效期

积分有效期（如：获得后12个月）。

</Steps>

## 积分操作

**手动调整积分**

管理员可以手动调整客户积分：
1. 进入客户详情页
2. 点击 **积分调整**
3. 输入调整数量和原因
4. 确认调整

<Callout type="warning">
积分调整会记录操作日志，用于审计追踪。
</Callout>

## 常见问题

<FAQ>
  <FAQItem question="积分过期了可以恢复吗？">
    过期积分无法恢复。建议在积分即将过期时通知会员使用。
  </FAQItem>
</FAQ>
