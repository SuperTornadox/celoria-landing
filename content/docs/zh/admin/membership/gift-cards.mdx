# 礼品卡

管理实体卡、电子卡和储值卡。

## 礼品卡类型

| 类型 | 说明 | 用途 |
|------|------|------|
| 实体卡 | 物理卡片 | 门店销售 |
| 电子卡 | 电子码 | 在线购买/赠送 |
| 储值卡 | 预充值会员卡 | 会员储值 |

## 创建礼品卡方案

<Steps>

### 1. 进入礼品卡管理

点击 **会员 > 礼品卡**。

### 2. 新建礼品卡方案

点击 **新建方案**。

### 3. 设置卡片信息

| 字段 | 说明 |
|------|------|
| 方案名称 | 如「节日礼品卡」 |
| 面值选项 | $50/$100/$200 等 |
| 有效期 | 购买后的有效天数 |
| 卡面设计 | 选择或上传卡面图片 |

### 4. 设置购买规则

- 是否可在线购买
- 购买数量限制
- 充值优惠（如充$100送$10）

### 5. 保存方案

</Steps>

## 销售礼品卡

**门店销售**
1. 在结账页面选择 **销售礼品卡**
2. 选择礼品卡方案和面值
3. 完成支付
4. 打印或发送电子卡给客户

**在线购买**
1. 客户在线上商城购买
2. 支付成功后自动发送电子礼品卡到邮箱
3. 客户可转赠给他人

## 使用礼品卡

1. 结账时客户提供礼品卡号
2. 员工输入卡号或扫码
3. 系统显示卡片余额
4. 确认使用金额
5. 完成抵扣

## 礼品卡查询

查看礼品卡的：
- 卡号和面值
- 当前余额
- 使用记录
- 有效期

## 常见问题

<FAQ>
  <FAQItem question="礼品卡丢失怎么办？">
    如果客户有购买记录，可以通过手机号或订单号查询并补发电子卡。实体卡遗失后只能挂失补办。
  </FAQItem>
</FAQ>
