# 服务配置

本文介绍如何在 Celoria 中配置和管理服务项目，包括创建服务、设置价格、配置服务组合等。

<Video id="V04" title="服务配置视频教程" duration="3-4 分钟" />

## 适用角色

- 超级管理员 (Super Admin)
- 管理员 (Admin)

## 前置条件

- 已登录管理员账号
- 拥有 `services:create` 或 `services:update` 权限

## 服务列表

进入 **服务** 菜单查看所有服务项目。

### 列表信息

| 列 | 说明 |
|------|------|
| 服务名称 | 服务显示名称 |
| 分类 | 所属服务分类 |
| 时长 | 预估服务时长 |
| 价格 | 服务价格 |
| 状态 | 启用/停用 |

### 筛选功能

- **分类筛选** - 按服务分类筛选
- **状态筛选** - 启用/停用/全部
- **搜索** - 按服务名称搜索

## 服务分类

在添加服务前，建议先设置服务分类。

### 创建分类

<Steps>

### 1. 进入分类管理

点击 **服务分类** 标签页。

### 2. 添加新分类

点击 **添加分类** 按钮。

### 3. 填写分类信息

| 字段 | 说明 |
|------|------|
| 分类名称 | 如「美甲」「美睫」「护理」 |
| 分类图标 | 选择或上传图标 |
| 排序 | 显示顺序 |

### 4. 保存分类

点击 **保存** 完成创建。

</Steps>

### 常用分类示例

- 美甲 (Manicure)
- 美足 (Pedicure)
- 美睫 (Eyelash)
- 美眉 (Eyebrow)
- 护理 (Treatment)
- 套餐 (Package)

## 添加服务

<Steps>

### 1. 点击添加服务

在服务列表页面右上角，点击 **添加服务** 按钮。

### 2. 填写基本信息

| 字段 | 说明 | 必填 |
|------|------|------|
| 服务名称 | 客户可见的服务名称 | 是 |
| 服务代码 | 内部标识代码 | 否 |
| 服务分类 | 选择所属分类 | 是 |
| 服务描述 | 详细描述，显示在预约页面 | 否 |

### 3. 设置时间和价格

| 字段 | 说明 |
|------|------|
| 服务时长 | 预估完成时间（分钟） |
| 服务价格 | 标准价格 |
| 会员价格 | 会员专享价（可选） |

<Callout type="info">
服务时长用于预约时间计算。建议设置略长于实际操作时间，预留过渡时间。
</Callout>

### 4. 配置可选项

**可选添加项 (Add-ons)**

可以设置该服务的可选附加项目：

- 加固 (+$5)
- 卸甲 (+$10)
- 特殊设计 (+$15)

**服务图片**

上传服务效果图，在预约页面展示。

### 5. 设置可服务员工

选择可以提供该服务的员工。默认所有员工可提供。

### 6. 启用服务

勾选 **启用** 选项，服务将在客户预约页面显示。

### 7. 保存服务

确认信息后点击 **保存**。

</Steps>

## 编辑服务

1. 在服务列表中点击目标服务
2. 进入服务详情页
3. 点击 **编辑** 按钮
4. 修改需要更新的信息
5. 点击 **保存**

### 批量调价

<Steps>

### 1. 选择服务

在服务列表中勾选需要调价的服务。

### 2. 点击批量调价

点击工具栏的 **批量调价** 按钮。

### 3. 设置调价方式

- **固定金额** - 每个服务加/减固定金额
- **百分比** - 按百分比上调/下调

### 4. 预览并确认

查看调价预览，确认后执行。

</Steps>

## 服务组合（套餐）

将多个服务打包成套餐，提供优惠价格。

### 创建套餐

<Steps>

### 1. 添加新服务

选择「套餐」分类。

### 2. 添加包含服务

在「包含服务」区域，添加套餐内的服务项目。

### 3. 设置套餐价格

套餐价格通常低于单项服务总价。

### 4. 设置套餐时长

系统会自动计算，也可手动调整。

</Steps>

### 套餐示例

| 套餐名称 | 包含服务 | 原价 | 套餐价 |
|----------|----------|------|--------|
| 豪华美甲套餐 | 基础美甲 + 手部护理 + 艺术设计 | $85 | $70 |
| 闺蜜双人套餐 | 2×基础美甲 | $70 | $60 |

## 服务状态

### 启用

服务正常显示，客户可预约。

### 停用

服务暂时下架，不显示在预约页面。

<Callout type="warning">
停用服务不会影响已有预约。如需取消已有预约，请单独处理。
</Callout>

### 删除

<Callout type="error">
有预约记录的服务不能删除，只能停用。这是为了保留历史数据完整性。
</Callout>

## 服务定价策略

### 门店差异定价

不同门店可以设置不同价格：

1. 进入服务详情页
2. 点击 **门店定价** 标签
3. 为每个门店设置价格
4. 未设置的门店使用默认价格

### 会员定价

为会员设置专享价格：

1. 在服务编辑页面
2. 勾选 **会员价格**
3. 输入会员专享价

## 常见问题

<FAQ>
  <FAQItem question="如何临时下架某个服务？">
    在服务列表中找到该服务，点击状态开关将其设为「停用」。客户将无法预约该服务，但已有预约不受影响。
  </FAQItem>
  <FAQItem question="可以设置不同员工的服务价格不同吗？">
    目前服务价格统一设置。如需按员工等级定价，建议创建不同版本的服务（如「高级美甲-资深技师」）。
  </FAQItem>
  <FAQItem question="如何导入现有服务列表？">
    在服务列表页面点击「导入」按钮，下载模板后按格式填写，然后上传即可批量导入。
  </FAQItem>
  <FAQItem question="套餐中的服务可以分开预约吗？">
    套餐作为一个整体预约，包含的服务会连续安排。如客户只需要部分服务，建议单独预约。
  </FAQItem>
</FAQ>
