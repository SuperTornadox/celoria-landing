# 门店访问范围

控制员工可以访问哪些门店的数据。

## 范围类型

**全局范围 (Global)**
- 可访问所有门店的数据
- 适用于：超级管理员、总部管理人员

**门店范围 (Store)**
- 只能访问指定门店的数据
- 适用于：门店经理、门店员工

**个人范围 (Self)**
- 只能访问自己的数据
- 适用于：普通员工查看自己的业绩

## 配置门店范围

1. 进入员工详情页
2. 点击 **访问范围** 标签
3. 选择范围类型
4. 如选择「指定门店」，勾选可访问的门店
5. 保存设置

## 常见问题

<FAQ>
  <FAQItem question="如何让员工只能看自己的业绩？">
    设置员工的数据范围为「个人范围」，这样在查看报表时只会显示自己的数据。
  </FAQItem>
</FAQ>
