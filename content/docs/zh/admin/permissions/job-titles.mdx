# 职位管理

职位是权限管理的核心单元，每个职位关联一组权限。

## 查看职位列表

进入 **管理 > 权限管理 > 职位** 查看所有职位。

## 创建职位

<Steps>

### 1. 点击添加职位

在职位列表页面点击 **添加职位**。

### 2. 填写职位信息

| 字段 | 说明 |
|------|------|
| 职位名称 | 如「资深美甲师」 |
| 基础角色 | 选择系统角色作为基础 |
| 职位描述 | 可选的说明文字 |

### 3. 配置权限

勾选该职位应有的权限。权限按模块分组显示：

**预约模块**
- appointments:view - 查看预约
- appointments:create - 创建预约
- appointments:update - 修改预约
- appointments:delete - 删除预约

**客户模块**
- guests:view - 查看客户
- guests:create - 创建客户
- guests:update - 修改客户

**服务模块**
- services:view - 查看服务
- services:manage - 管理服务

**报表模块**
- reports:view - 查看报表
- reports:export - 导出报表

### 4. 设置门店范围

选择该职位可访问的门店范围：

- **全部门店** - 可访问所有门店
- **指定门店** - 只能访问选定的门店
- **所属门店** - 只能访问自己被分配的门店

### 5. 保存职位

点击 **保存** 完成创建。

</Steps>

## 编辑职位

1. 点击职位名称进入详情
2. 点击 **编辑** 按钮
3. 修改权限配置
4. 保存更改

<Callout type="warning">
修改职位权限会立即影响所有该职位的员工。建议在非营业时段进行权限调整。
</Callout>

## 常见问题

<FAQ>
  <FAQItem question="员工反映看不到某个菜单怎么办？">
    检查该员工的职位权限配置，确认是否有对应模块的 view 权限。如有需要，可在职位管理中添加相应权限。
  </FAQItem>
  <FAQItem question="可以给单个员工特殊权限吗？">
    可以。在员工详情页的「权限」标签中，可以为该员工单独添加或移除权限，这会覆盖职位默认权限。
  </FAQItem>
</FAQ>
