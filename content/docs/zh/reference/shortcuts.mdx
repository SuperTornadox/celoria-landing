# 快捷键

本文档介绍 Celoria 的快捷键和操作技巧，帮助你提高工作效率。

## 全局快捷键

这些快捷键在系统的任何页面都可以使用。

| 快捷键 | 功能 | 说明 |
|--------|------|------|
| `Ctrl/Cmd + K` | 全局搜索 | 打开搜索框，快速查找客人、预约、员工等 |
| `Ctrl/Cmd + /` | 快捷键帮助 | 显示当前页面可用的快捷键 |
| `Esc` | 关闭弹窗 | 关闭当前打开的对话框或模态框 |
| `Ctrl/Cmd + R` | 刷新页面 | 重新加载当前页面数据 |

## 导航快捷键

快速切换到不同页面。

| 快捷键 | 功能 |
|--------|------|
| `G + D` | 前往仪表板 (Dashboard) |
| `G + A` | 前往预约 (Appointments) |
| `G + G` | 前往客人 (Guests) |
| `G + E` | 前往员工 (Employees) |
| `G + S` | 前往服务 (Services) |
| `G + R` | 前往报表 (Reports) |

<Callout type="info">
导航快捷键需要先按 G 键，然后快速按下第二个键。
</Callout>

## 操作技巧

### 快速搜索

使用 `Ctrl/Cmd + K` 打开全局搜索，可以快速查找：
- 客人姓名或手机号
- 预约编号
- 员工姓名
- 服务名称

### 键盘导航

尽量使用键盘完成操作，减少鼠标切换：
1. 使用 `Tab` 在字段间移动
2. 使用 `Enter` 确认选择
3. 使用快捷键切换页面

### 批量操作

选择多个项目后可以批量操作：
1. 按住 `Shift` 点击可选择连续项目
2. 按住 `Ctrl/Cmd` 点击可选择多个不连续项目
3. 使用 `Ctrl/Cmd + A` 全选

### 右键菜单

在大多数列表项上右键可以显示快捷操作菜单：
- 预约：查看、编辑、取消、开始服务
- 客人：查看、编辑、新建预约
- 员工：查看、编辑排班

## 自定义快捷键

部分快捷键可以在设置中自定义：

1. 进入 **设置 > 个人偏好**
2. 找到 **快捷键设置**
3. 点击要修改的快捷键
4. 按下新的组合键
5. 保存设置

<Callout type="warning">
自定义快捷键时注意避免与浏览器或系统快捷键冲突。
</Callout>

## 常见问题

<FAQ>
  <FAQItem question="快捷键不起作用怎么办？">
    1. 确保焦点在正确的区域（不在输入框中）
    2. 检查是否与浏览器快捷键冲突
    3. 尝试刷新页面后重试
  </FAQItem>
  <FAQItem question="如何查看当前页面的快捷键？">
    按 `Ctrl/Cmd + /` 可以显示当前页面所有可用的快捷键。
  </FAQItem>
  <FAQItem question="可以禁用快捷键吗？">
    在设置中可以关闭快捷键功能，或者只禁用特定的快捷键。
  </FAQItem>
</FAQ>
